# HostelCart AI

HostelCart AI is a collaborative group-ordering platform designed for hostel students.

Students can create or join active group orders, add items they want to purchase, split delivery charges, and collectively meet minimum order requirements on platforms like Blinkit and Instamart.

## Features

- User Authentication
- Create Group Orders
- Join Existing Orders
- Add and Manage Items
- Split Delivery Charges Automatically
- Real-Time Order Updates
- AI-Based Order Recommendations
- ML-Based Order Completion Prediction
